# Mediterranean diet adherence is associated with antiviral, neuroimmune, and cardiometabolic proteomic profiles in people with HIV

> Data repository accompanying the manuscript: **"Mediterranean diet adherence is associated with antiviral, neuroimmune, and cardiometabolic proteomic profiles in people with HIV"**  
> Mónica Manzano*, Elena Moreno, Laura Martín-Pedraza, Claudio Díaz-García, Santiago Moreno, Beatriz Navia and Sergio Serrano-Villar* — *Frontiers in Nutrition*, 2026 (doi: )  
> (*) Corresponding authors
---

## Contents

| File | Description |
|------|-------------|
| `dataset_proteins.csv` | Protein measurements and covariables for all study samples |

## Variable Description

| Column | Type | Description |
|--------|------|-------------|
| `id` | string | Unique sample identifier |
| `ACTN4` | numeric | NPX value for protein ACTN4 |
| `...` | numeric | NPX value for protein ... |
| `YTHDF3` | numeric | NPX value for protein YTHDF3 |
| `med-dqi_group` | categorical | Grouped version of `med-dqi` (0 = low adherence, 1 = high adherence) |
| `med-dqi` | numeric | Mediterranean Diet Quality Index (continuous score from 0 to 14) |
| `bmi` | numeric | Body mass index (kg/m²) |
| `cd4` | numeric | CD4+ T-cell count (cells/µL) |
| `ethnicity` | categorical | Self-reported ethnicity (Caucasian / Latin American / Sub-Saharan African) |
| `sex` | categorical | Biological sex (0 = Female, 1 = Male) |
| `age` | numeric | Age at time of sample collection (years) |

## Usage

This dataset is provided as-is to support reproducibility of the results reported in the manuscript.
